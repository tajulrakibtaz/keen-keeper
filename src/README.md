Project Name: Keen Keeper

Description:
Keen Keeper is a web application where you can explore your connections with your friends in a simple and interactive way. On the website, you can click on a friend’s card to see detailed information about that friend and understand how well you are connected. There are also buttons like call, video, and text. When you click on them, the activity is recorded and shown on the timeline page as history. Additionally, there is a stats section where you can see a pie chart showing your usage of calls, video, and text.

Technologies Used:
React.js, Tailwind CSS, React Icons, Recharts

Key Features:

Clickable friend cards to view connection details.
Timeline page that tracks call, video, and text history.
Stats section with a pie chart showing communication activity.
